Using deep learning to recognise, optimise, and generate images. [In progress]

Introduction:

We will be using the mnist dataset to recongise, optimise and generate handwritten digits. I used the programming language Python to create this project. In the practical world this could be used to digitalise handwritten notes or convert digital text to handwritten notes.

Recognition:

To classify our handwritten digits we will use conversational neural networks (CNN). 60,000 images are used to train our network, remaining 10,000 which can be used to calculate he accuracy of our model. I compared adapative moment estimation (ADAM) to stochastic gradient descent for being used as the model's optimizer and as theorised I conluded ADAM was the more accurate option, increasing accuracy by 0.4%.
